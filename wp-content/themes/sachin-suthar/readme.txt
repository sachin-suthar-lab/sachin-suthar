A premium Full Site Editing block theme for the Sachin Suthar developer portfolio. Designed to pair with the Smart Blocks plugin.

== Description ==
Dark, minimal aesthetic. Inter typography. Glassmorphism, subtle gradients, and a brand palette tuned for product/agency portfolios. Templates: front-page, page, single, archive, index, 404. Sticky glass header with anchor navigation; surface-elevated footer with link columns.

Activate the **Smart Blocks** plugin alongside this theme to get the 9 portfolio sections (hero, services, skills, experience, tech stack, projects, testimonials, CTA, contact).
